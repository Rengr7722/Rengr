Patching Notes: ( Same as video above)

1. Run Flow3d hydro 2023R2U1 Installer. Select install license server and flow-3d hydro. Check the box "opt out of RSS Feed". If it ask to stop existing server just select skip.
2. Run Flow3d Post 2023R2 Installer. Enter the Ip address 127.0.0.1.
3. Place patcher in flow3d hydro install directory for example. "C:\flow3d\HYDRO_2023R2u1",   and run patcher. Click on Patch flow-3d Hydro.
4. Place patcher in flow3d Post install directory for example. "C:\flow3d\POST_2023R2",         and run patcher. Click on Patch flow-3d Post.
5. Restart Computer
6. If you get license error, make sure the service "HYDRO_2023R2" is running in task manager->Services.
